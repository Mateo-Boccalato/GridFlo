What it does

Small language, visible flow

01

Wire Data

Inputs, constants, transforms, and outputs are connected directly in `.gf` files.

02

Run Graphs

The interpreter parses the grid into a typed graph and evaluates values as they are needed.

03

Reuse Plates

Package a mini graph as a `plate`, then call it like any other transform cell.

Start coding

No installs. Just Python.

GridFlo has no third-party dependencies. Clone the project, open a `.gf` sample, and run it with the command line entry point. 

python --version
python gridflow.py check "sample programs/fizzbuzz.gf"
python gridflow.py run "sample programs/fizzbuzz.gf" n=15
